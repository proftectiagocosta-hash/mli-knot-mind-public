# Limites Públicos

Este repositório é uma vitrine pública sanitizada.

## Permitido

- explicar conceitos;
- documentar princípios;
- mostrar exemplos fictícios;
- apresentar casos de uso;
- publicar arquitetura pública;
- demonstrar diferença entre resposta genérica e resposta governada.

## Não permitido

- publicar prompts privados completos;
- publicar checkpoints reais;
- publicar histórico bruto de conversas;
- expor rotinas internas sensíveis;
- importar automaticamente material do núcleo privado;
- publicar dados pessoais;
- publicar comandos privados de ativação;
- confundir vitrine pública com núcleo operacional.

## Regra de ouro

Se um conteúdo permite reconstruir o núcleo privado ou expõe contexto sensível, ele não pertence a esta vitrine.
