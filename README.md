## Português

### O que é

O **MLI-Knot Mind** é um framework de governança e processamento criado para melhorar a qualidade de respostas em fluxos assistidos por IA.

Ele não é memória persistente, não é banco de dados de estados, não é cofre de histórico e não é executor automático.

Sua função é atuar como uma **camada de processamento** que orienta como o contexto deve ser lido, separado, filtrado e transformado em resposta útil, honesta e não genérica.

### Princípios públicos

- honestidade intelectual;
- separação entre fato, hipótese, inferência e limitação;
- rejeição de improviso genérico;
- clareza de escopo;
- disciplina de contexto;
- rastreabilidade mínima;
- governança de prompts;
- anti-alucinação;
- resposta útil antes de resposta impressionante.

### O que este repositório contém

- visão pública do framework;
- princípios conceituais;
- arquitetura pública;
- exemplos sanitizados;
- casos de uso;
- limites públicos;
- documentação bilíngue;
- vitrine HTML simples.

### O que este repositório não contém

- prompts privados completos;
- checkpoints reais;
- histórico bruto;
- material operacional sensível;
- estados internos do ecossistema;
- dados pessoais;
- regras privadas de ativação;
- mecanismos internos completos.

### Objetivo

A proposta desta vitrine é apresentar o **MLI-Knot Mind** como uma abordagem de qualidade para uso de IA, especialmente em contextos que exigem:

- raciocínio disciplinado;
- respostas úteis;
- transparência de incerteza;
- redução de alucinação;
- governança de contexto;
- continuidade com integridade;
- separação entre execução real e intenção declarada.

## English

### What it is

**MLI-Knot Mind** is a governance and processing framework designed to improve the quality of responses in AI-assisted workflows.

It is not persistent memory, not a state database, not a raw-history vault, and not an automatic executor.

Its function is to act as a **processing layer** that guides how context should be read, separated, filtered, and transformed into useful, honest, and non-generic responses.

### Public principles

- intellectual honesty;
- separation between fact, hypothesis, inference, and limitation;
- rejection of generic improvisation;
- scope clarity;
- context discipline;
- minimal traceability;
- prompt governance;
- anti-hallucination;
- useful answers before impressive answers.

### What this repository contains

- public framework overview;
- conceptual principles;
- public architecture;
- sanitized examples;
- use cases;
- public limits;
- bilingual documentation;
- simple HTML showcase.

### What this repository does not contain

- full private prompts;
- real checkpoints;
- raw history;
- sensitive operational material;
- internal ecosystem states;
- personal data;
- private activation rules;
- complete internal mechanisms.

### Purpose

This showcase presents **MLI-Knot Mind** as a quality-oriented approach to AI usage, especially in contexts that demand:

- disciplined reasoning;
- useful responses;
- transparent uncertainty;
- reduced hallucination;
- context governance;
- continuity with integrity;
- separation between real execution and declared intention.
